# RishuBoard

A simple and good looking hackpad which have 15 keys and one rotary encoder. Every key will be given different task or shortcut. I've used XIAO RP2040 microcontroller, 2 RGB Leds for activety and status view. Rotary encoder will be used for adjusting the volume of main system. I've made the case in two parts "top" and "bottom" I thought to add top plate as well buy I didn't because I wanted to make to look like a coustom build project or else people would think that I've bought it. The designe I made is pretty simple but good at the same time.

I've designed this hackpad for my younger brother and name it after him "RishuPad". After I'll finish building this then I'll gift this hackpad to him. 

Here's some images of the project and BOM in bottom. I have added the diodes in BOM but didn't added it's cost because I've alr some in my home and I'll be using that in this project and for the 3D case I'll get it from printing-legion.
